# 🏅 Olympic Games Historical Analysis (1900-2000) - Power BI Project

## 📝 Project Overview
This Power BI dashboard provides a comprehensive historical analysis of the Olympic Games over a century (1900–2000). The project focuses on visualizing medal trends, demographic distributions, and sports popularity to uncover deep insights into how the games have evolved over time.<img width="659" height="366" alt="Screenshot 2026-06-15 003008" src="https://github.com/user-attachments/assets/dd17daf6-4aed-435c-a31c-b59d59b167e7" />


---

## 💡 Key Insights Delivered
- **Massive Achievement:** Tracked over **32.5K total medals** distributed over the 100-year span.
- **Participation Trends:** Analyzed the dynamic historical shift and growth in medal counts, highlighting the trends between male and female athletes over the decades.
- **Seasonal Distribution:** Visualized the dominant share of **Summer Games (78.91%)** compared to **Winter Games (21.09%)**.
- **Sports Popularity:** Identified **Athletics, Skiing, and Aquatics** as the top sports by the total number of tournaments.

## 🛠️ Technical Skills Demonstrated
- **Advanced Data Visualization:** Built a cohesive, custom grey and lime-themed dashboard with optimal visual hierarchy.
- **Time-Series Analysis:** Designed advanced line and area charts (`Medal Trends by Year`) to display complex multi-metric changes (Medal counts vs. Men/Women distribution) over a 100-year timeline.
- **Interactive UX/UI Navigation:** Implemented custom side-navigation buttons (`Overview & Trends`, `Countries & Medals`, `Sport & Demographic`) to ensure seamless storytelling and user experience.
- **Data Customization:** Handled large historical datasets, ensuring proper filtering and clean visual aggregation.
